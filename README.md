# avito-anti-booked

Tampermonkey-скрипт для **Avito.ru**, который автоматически находит карточки товаров с пометкой **«Забронировано»** и скрывает их (или подсвечивает красным).

## ✨ Особенности
- 🔍 Ищет пометку «Забронировано» независимо от автогенерируемых CSS-классов
- ⚡ Работает с динамической подгрузкой (SPA) без перезагрузки страницы
- 🛡️ **Не ломает вёрстку**: использует безопасное «схлопывание» элемента вместо `display: none`
- 🎛️ Переключаемый режим: полное скрытие или красная подсветка
- 🌐 Совместим с Tampermonkey, Violentmonkey, Greasemonkey (Firefox, Chrome, Edge, Safari)

## 📦 Установка
1. Установите расширение **Tampermonkey** (или Violentmonkey) в ваш браузер.
2. В настройках **Tampermonkey** дайте доступ на выполнение пользовательских скриптов
3. Откройте вкладку «Создать новый скрипт».
4. Скопируйте код скрипта avito-anti-booked.js и вставьте в редактор.
5. Нажмите **Сохранить**.
6. Перейдите на `avito.ru` — скрипт начнёт работать автоматически.
7. Внизу страницы появится кнопка Скрыть/Показать


## 🔧 Как работает
- Скрипт использует `MutationObserver` для отслеживания изменений в DOM.
- При появлении нового контента он ищет `<span>` с точным текстом «Забронировано».
- Находит родительский контейнер карточки по атрибуту `data-marker="item"`.
- Применяет безопасные стили для скрытия (`height: 0`, `overflow: hidden`, `opacity: 0`) или красную рамку.
- Помечает обработанные карточки, чтобы избежать дублирования и лишней нагрузки на браузер.

## ⚠️ Дисклеймер
Скрипт не является официальным продуктом Avito и не связан с ним. Модифицирует только клиентскую часть браузера. Используйте на свой страх и риск.
